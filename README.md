# HeiAdmin

## 项目介绍

学习 vue3，搭建的一个后台管理系统

## 安装项目依赖

```
npm install
```

### 本地开发环境启动项目命令，默认打开 http://localhost:4000，
**登录账号/密码：admin/123456**

```
npm run dev
```

### 区分环境打包，默认开发环境打包

```
npm run build:dev
```
